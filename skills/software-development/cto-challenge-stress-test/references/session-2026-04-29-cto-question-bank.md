# Session reference: CTO question bank consolidation (2026-04-29)

Use this note when a user asks to merge or dedupe multiple CTO interview-question sources into one canonical bank.

## What happened in this session
- The user first evaluated a HelloSky CTO interview set, then asked to combine it with an earlier 50-question bank.
- The earlier 50-question wording was not fully preserved in live context, so the bank was reconstructed at the theme level.
- The user then supplied a Workable CTO interview page, which added a small number of overlapping questions and one clearly new theme: staying current with technological trends / learning resources.

## Consolidation rules that worked well
1. Prefer a canonical theme over verbatim duplicates when two questions test the same CTO behavior.
2. Keep one representative phrasing per theme.
3. Preserve provenance by noting which source pages contained the theme.
4. Mark exact duplicates and near-duplicates separately if asked.
5. If the live context does not contain the exact historic bank, say so and present a reconstructed equivalent rather than pretending exact recall.
